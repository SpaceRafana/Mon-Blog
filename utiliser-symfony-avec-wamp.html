Tutoriel d'utilisation de Symfony avec Wamp

Première étape : Mettre le projet Symfony dans le dossier " www "

Dans le dossier d’installation de Wamp se trouve un dossier qui se nomme « www ». Ce dossier qui doit contenir tout vos sites en local avec Wampserver que vous le connaissez peut-être déjà.

Alors, mettez le projet symfony dedans. Exemple E:/wamp64/www/

Deuxième étape : Créer un virtual host

Dans la page d’accueil Wamp par localhost. Cliquez le menu " Ajouter un Virtual Host ".

Après avoir entré dans la page Virtual Host, il y a deux formulaires marqué Requis à remplir :

" Nom du Virtual Host " : nom de votre virtual host selon votre choix (ex : mon-site.com).
" Chemin complet absolu du dossier " : entrez le chemin complet qui pointe vers l’index.php de votre projet Symfony (ex : E:/wamp64/www/mon-projet-symfony/public/).

Dernière étape : Redémarrer Wamp

Après avoir redémarrer wamp. Accéder à votre projet/site par le nom de virtual host que vous avez choisi.

Remarque :

# N’oubliez pas d’entrer « http:// » avant le nom de votre virtual host dans l’url.
# Il n’est pas obligatoire de faire un symfony server:start dans le terminal comme d’habitude pour lancer symfony, mais juste seulement démarrer wamp et entrer le nom de virtual host dans l’url
